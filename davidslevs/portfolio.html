zombie_game_ FPS Framework

For this one, I made the entire gun system from scratch. It features decent lag compensation with little to no compromises, along with complete support for visual effects and replication, along with some other small touches.

The features include:

  • Viewmodel
  • Procedural effects like swaying and bobble
  • Viewmodel replication (arm movements and head)
  • Hit registration
  • Hit verification
  • Lag compensation
  • Muzzle flash support
  • Shell ejection support
  • Secure hit registration and reload cooldowns

.. and much more. It's created with object-oriented programming, so it's easy to drag and drop and create new guns from just a settings entry.

Note: I did not create the models or animations myself. I commissioned those.

2D destructible terrain in roblox

This is a work-in-progress physics system for a worms-like game I'm making on roblox.

It uses the CanvasDraw module to write pixels to the screen.

It features:

  • Circle to pixel collisions
  • Destructible terrain
  • Pixel perfect physics system
  • Custom particles system
  • Smooth 2D camera
